# 📊 Maven Market Sales Analysis Dashboard | Power BI

## Overview

This project is an interactive **Power BI Sales Dashboard** developed using the **Maven Market** retail dataset. The dashboard provides comprehensive business insights into **sales performance, profit analysis, customer behavior, product performance, and store operations** through dynamic visualizations and KPI reporting.

The project demonstrates end-to-end **Business Intelligence (BI)** skills, including **data cleaning, data modeling, DAX calculations, dashboard design, and data visualization** to support data-driven business decisions.
